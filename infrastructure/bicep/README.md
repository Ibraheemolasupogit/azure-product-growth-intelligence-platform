# Bicep Skeleton

This folder is reserved for a future Bicep implementation. Milestone 12 provides architecture guidance only.

Potential future modules:

- resource group and tagging conventions;
- ADLS Gen2 storage accounts and containers;
- Event Hubs namespace;
- Data Factory or Synapse orchestration;
- Azure ML workspace;
- Key Vault;
- Azure Monitor and Application Insights;
- Microsoft Purview registration notes;
- Power BI workspace guidance outside Bicep.

No Bicep deployment file is wired into CI, and no live Azure deployment command is included.
