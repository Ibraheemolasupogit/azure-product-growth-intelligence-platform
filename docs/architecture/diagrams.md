# Architecture Diagrams

The diagrams in `diagrams/` are Mermaid-compatible source files intended for readable architecture review. They are not generated deployment diagrams and do not imply live Azure resources exist.

| Diagram | Purpose |
| --- | --- |
| `diagrams/azure-reference-architecture.mmd` | Shows the Azure-mappable service architecture. |
| `diagrams/local-data-flow.mmd` | Shows local deterministic data flow. |
| `diagrams/ml-and-analytics-lifecycle.mmd` | Shows analytics, ML, insight, and reporting lifecycle. |
| `diagrams/governance-lineage-flow.mmd` | Shows governance, lineage, and evidence flow. |

## Review Notes

Use these diagrams alongside `final-azure-reference-architecture.md` and `local-to-azure-mapping.md`. The diagrams are intentionally compact so they can be discussed in interviews or architecture reviews without visual clutter.
