# Technical Review Guide

## Data Engineering Review

Inspect ingestion, validation, contracts, quarantine behavior, and Milestone 3 evidence.

## Analytics Review

Inspect funnel, retention, experiment analysis, metric framework, and reporting semantic model.

## ML Review

Inspect churn, segmentation, recommendation baselines, leakage controls, model cards, and tests.

## AI Review

Inspect deterministic product insight generation, prompt package behavior, governance checks, and no-live-call defaults.

## Azure Architecture Review

Inspect final architecture docs, local-to-Azure mapping, diagrams, infrastructure skeleton README files, and CI boundaries.

## Commands

```bash
make quality
make verify-final-evidence
make verify-portfolio
```
