# Portfolio Walkthrough

Start with the README, then review the architecture and evidence folders in order.

## What Each Milestone Proves

- Milestones 1-2: repository foundation and deterministic synthetic NexaFlow data.
- Milestone 3: governed ingestion, contracts, quality, quarantine, and lineage.
- Milestones 4-5: funnel analytics and retention/cohort analysis.
- Milestones 6-8: churn, segmentation, and recommendation ML workflows.
- Milestone 9: governed experiment analysis with guardrails and power checks.
- Milestone 10: deterministic product insight assistant grounded in evidence.
- Milestone 11: Power BI-ready semantic reporting layer.
- Milestone 12: final Azure architecture, portfolio review docs, and deployment options.

## Reviewer Path

1. Read `README.md`.
2. Open `docs/architecture/final-azure-reference-architecture.md`.
3. Inspect `docs/evidence/README.md`.
4. Run `make quality`.
5. Run `make verify-final-evidence`.
6. Review `docs/portfolio/technical-review-guide.md` for role-specific routes.
