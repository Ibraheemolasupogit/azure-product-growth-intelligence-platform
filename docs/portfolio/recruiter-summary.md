# Recruiter Summary

This repository demonstrates a complete production-style local reference implementation for product analytics and Azure-mappable growth intelligence.

In plain language, it shows how a product team could move from synthetic product events through quality validation, funnel analysis, retention, churn prediction, segmentation, recommendations, experiments, governed product insights, and Power BI-ready reporting.

It is strongest for roles involving product data science, growth analytics, analytics engineering, ML engineering, AI engineering, and Azure data architecture. It is not a live deployed Azure product.
